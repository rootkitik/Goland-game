Первая версия консольной чат-игры "Угадай слово". В программу были реализованны: 

1. 🎲 Генерация случайного числа от 1 до 100
2. 🔁 Расширенный выбор после каждого завершения цикла (победы/поражения)
3. 📜 Лог изменений (версия Go, компилятор, список функционала)
4. 🖼 ASCII-арт при запуске
5. 🧭 Меню навигации: игра / лог / выход / возврат в начало
